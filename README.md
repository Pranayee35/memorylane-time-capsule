## Problem
Preserving memories in a meaningful, time-locked way.

## Solution
Users create digital capsules with text and media that unlock in the future.

## Features
- Create time capsules with images/videos
- Secure media storage (Cloudinary)
- Time-based locking & countdown
- Dynamic dashboard
- Capsule details view

## Tech Stack
- Frontend: React, Tailwind, React Router
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Media: Cloudinary

## How to Run
1. `cd server && npm run dev`
2. `cd client && npm run dev`
3. Open `http://localhost:5173`
